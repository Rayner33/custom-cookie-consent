A simple cookie consent plugin for WordPress.

== Description ==
This plugin provides a customizable cookie consent banner for your WordPress site, ensuring compliance with GDPR and other privacy laws.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the settings under the 'Cookie Consent' menu in the admin panel.
